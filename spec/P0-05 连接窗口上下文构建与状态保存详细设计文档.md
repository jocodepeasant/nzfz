## 1. 版本说明

### 1.1 v1.1 修订目标

本文档用于定义 `WindowManager.connect_window()` 中的**连接窗口上下文构建阶段**。

P0-05 的职责是：

```text
在 P0-04 连接前校验通过后，重新读取目标窗口当前真实状态，构建 ConnectedWindow 候选对象。
```

本阶段只构建 `ConnectedWindow` 对象，不直接保存为当前连接状态。

真正写入：

```python
self._connected_window
```

必须发生在：

```text
P0-06 激活目标窗口成功之后。
```

---

## 2. 与整体连接流程的关系

完整连接流程为：

```text
P0-04：连接前校验
    ↓
P0-05：构建 ConnectedWindow 候选对象
    ↓
P0-06：激活目标窗口
    ↓
保存 _connected_window
    ↓
返回 ConnectResult.success
```

P0-05 位于连接前校验之后、窗口激活之前。

---

## 3. 设计目标

P0-05 的目标是根据目标窗口当前真实状态构建：

```python
ConnectedWindow
```

需要写入字段：

```text
hwnd
title
process_name
pid
window_rect
client_rect
dpi_scale
connected_at
```

其中：

- `hwnd` 来自 `WindowInfo.hwnd`；
- `title` 必须重新读取；
- `pid` 必须重新读取并再次校验；
- `process_name` 必须重新读取；
- `window_rect` 必须重新读取；
- `client_rect` 必须重新读取并转换为屏幕坐标；
- `dpi_scale` 当前固定为 `1.0`；
- `connected_at` 由模型默认工厂生成。

## 5. 模块边界

### 5.1 P0-05 负责内容

P0-05 负责：

- 接收已通过 P0-04 校验的 `WindowInfo`；
- 根据 `WindowInfo.hwnd` 重新读取当前窗口状态；
- 防御性检查窗口句柄是否仍有效；
- 防御性检查窗口是否仍可见；
- 防御性检查窗口是否最小化；
- 重新获取当前窗口标题；
- 校验当前标题非空；
- 重新获取当前窗口 PID；
- 校验当前 PID 与 `WindowInfo.pid` 一致；
- 重新获取当前进程名；
- 获取当前窗口完整矩形；
- 获取当前客户区矩形；
- 将客户区矩形转换为屏幕坐标；
- 校验窗口完整矩形尺寸；
- 校验客户区矩形尺寸；
- 获取 DPI 缩放比例；
- 构造 `ConnectedWindow` 候选对象；
- 构建失败时返回明确错误；
- 构建失败时不覆盖旧连接状态。

- 激活目标窗口；
- 保存 `_connected_window`；
- 保存 `_last_window_info`；

- 健康检测；

- 多窗口会话管理；

## 6. 核心语义

### 6.1 `ConnectedWindow` 是连接候选上下文

P0-05 构建出来的 `ConnectedWindow` 表示：

```text
目标窗口在当前时刻可用于连接的候选上下文。
```

但它还不是最终连接状态。

只有 P0-06 激活成功后，它才可以被保存为：

```python
self._connected_window
```

### 6.2 不信任 P0-04 的读取结果

虽然 P0-04 已经校验过窗口状态，但窗口状态可能在极短时间内变化。

因此 P0-05 仍然需要进行防御性重新读取和校验。

### 6.3 最小化窗口必须提前失败

P0-05 中获取矩形之前必须再次检查：

```python
win32gui.IsIconic(hwnd)
```

如果窗口已最小化，立即失败，不继续读取窗口矩形和客户区矩形。

原因：

- 最小化窗口的 `GetWindowRect` 可能返回异常占位尺寸；
- 客户区坐标可能不可靠；
- 错误原因应明确提示用户恢复窗口。

## 7. 常量设计

建议在 `window_manager.py` 中统一定义：

```python
MIN_WINDOW_WIDTH = 100
MIN_WINDOW_HEIGHT = 100

MIN_CLIENT_WIDTH = 100
MIN_CLIENT_HEIGHT = 100
```

说明：

| 常量                | 用途                 |
| ------------------- | -------------------- |
| `MIN_WINDOW_WIDTH`  | 完整窗口矩形最小宽度 |
| `MIN_WINDOW_HEIGHT` | 完整窗口矩形最小高度 |
| `MIN_CLIENT_WIDTH`  | 客户区矩形最小宽度   |
| `MIN_CLIENT_HEIGHT` | 客户区矩形最小高度   |

即使当前数值相同，也应保持语义分离。

## 9. 上下文构建流程

### 9.1 标准流程

推荐顺序：

```text
1. 获取 hwnd
2. 防御性调用 IsWindow(hwnd)
3. 防御性调用 IsWindowVisible(hwnd)
4. 防御性调用 IsIconic(hwnd)
5. 调用 GetWindowText(hwnd)
6. 校验标题非空
7. 调用 GetWindowThreadProcessId(hwnd)
8. 校验当前 PID 与 WindowInfo.pid 一致
9. 通过 psutil.Process(pid) 获取进程对象
10. 获取 process.name()
11. 调用 GetWindowRect(hwnd)
12. 构建 window_rect
13. 校验 window_rect 尺寸
14. 调用 GetClientRect(hwnd)
15. 使用 ClientToScreen 转换客户区左上角
16. 使用 ClientToScreen 转换客户区右下角
17. 构建 client_rect
18. 校验 client_rect 尺寸
19. 获取 dpi_scale，当前固定为 1.0
20. 构建 ConnectedWindow
21. 返回构建成功
```

## 10. 窗口矩形设计

### 10.1 获取方式

```python
left, top, right, bottom = win32gui.GetWindowRect(hwnd)
```

构造：

```python
window_rect = WindowRect(
    left=left,
    top=top,
    right=right,
    bottom=bottom,
)
```

### 10.2 尺寸校验

```python
if window_rect.width < MIN_WINDOW_WIDTH:
    return False, "连接失败：窗口宽度过小", None

if window_rect.height < MIN_WINDOW_HEIGHT:
    return False, "连接失败：窗口高度过小", None
```

---

## 11. 客户区矩形设计

### 11.1 获取方式

客户区局部坐标：

```python
client_left, client_top, client_right, client_bottom = win32gui.GetClientRect(hwnd)
```

通常 `client_left`、`client_top` 为 `0`，但实现不应强依赖这一点。

转换为屏幕坐标：

```python
screen_left, screen_top = win32gui.ClientToScreen(
    hwnd,
    (client_left, client_top),
)

screen_right, screen_bottom = win32gui.ClientToScreen(
    hwnd,
    (client_right, client_bottom),
)
```

构造：

```python
client_rect = WindowRect(
    left=screen_left,
    top=screen_top,
    right=screen_right,
    bottom=screen_bottom,
)
```

### 11.2 坐标系约束

`client_rect` 必须使用屏幕坐标。

禁止将 `GetClientRect()` 返回的客户区局部坐标直接保存到 `ConnectedWindow.client_rect`。

### 11.3 尺寸校验

```python
if client_rect.width < MIN_CLIENT_WIDTH:
    return False, "连接失败：客户区宽度过小", None

if client_rect.height < MIN_CLIENT_HEIGHT:
    return False, "连接失败：客户区高度过小", None
```

## 13. 错误处理设计

### 13.1 失败错误信息

| 场景               | 错误信息建议                                |
| ------------------ | ------------------------------------------- |
| 句柄失效           | `连接失败：窗口句柄已失效，请重新搜索`      |
| 窗口不可见         | `连接失败：窗口不可见`                      |
| 窗口最小化         | `连接失败：窗口已最小化，请恢复窗口后重试`  |
| 标题为空           | `连接失败：窗口标题为空，请重新搜索`        |
| PID 不一致         | `连接失败：窗口进程已变化，请重新搜索`      |
| 进程不存在         | `连接失败：目标进程不存在`                  |
| 进程不可访问       | `连接失败：无法访问目标进程信息`            |
| 进程名为空         | `连接失败：无法获取目标进程名称`            |
| 获取窗口矩形失败   | `连接失败：获取窗口矩形失败`                |
| 窗口宽度过小       | `连接失败：窗口宽度过小`                    |
| 窗口高度过小       | `连接失败：窗口高度过小`                    |
| 获取客户区矩形失败 | `连接失败：获取客户区矩形失败`              |
| 客户区宽度过小     | `连接失败：客户区宽度过小`                  |
| 客户区高度过小     | `连接失败：客户区高度过小`                  |
| 未知异常           | `连接失败：构建连接上下文异常：{exception}` |

### 13.2 失败状态策略

P0-05 构建失败时：

- 不修改 `_connected_window`；
- 不修改 `_last_window_info`；
- 由 `connect_window()` 主流程写入 `_last_error`；
- 返回失败结果。

## 14. 推荐伪代码

```python
def _build_connected_window(
    self,
    window: WindowInfo,
) -> tuple[bool, str, Optional[ConnectedWindow]]:
    try:
        hwnd = window.hwnd

        if not win32gui.IsWindow(hwnd):
            return False, "连接失败：窗口句柄已失效，请重新搜索", None

        if not win32gui.IsWindowVisible(hwnd):
            return False, "连接失败：窗口不可见", None

        if win32gui.IsIconic(hwnd):
            return False, "连接失败：窗口已最小化，请恢复窗口后重试", None

        title = win32gui.GetWindowText(hwnd).strip()
        if not title:
            return False, "连接失败：窗口标题为空，请重新搜索", None

        _, current_pid = win32process.GetWindowThreadProcessId(hwnd)
        if current_pid != window.pid:
            return False, "连接失败：窗口进程已变化，请重新搜索", None

        try:
            process = psutil.Process(current_pid)
            process_name = process.name()
            if not process_name:
                return False, "连接失败：无法获取目标进程名称", None
        except psutil.NoSuchProcess:
            return False, "连接失败：目标进程不存在", None
        except psutil.AccessDenied:
            return False, "连接失败：无法访问目标进程信息", None
        except psutil.ZombieProcess:
            return False, "连接失败：目标进程状态异常", None

        left, top, right, bottom = win32gui.GetWindowRect(hwnd)
        window_rect = WindowRect(left, top, right, bottom)

        if window_rect.width < MIN_WINDOW_WIDTH:
            return False, "连接失败：窗口宽度过小", None

        if window_rect.height < MIN_WINDOW_HEIGHT:
            return False, "连接失败：窗口高度过小", None

        c_left, c_top, c_right, c_bottom = win32gui.GetClientRect(hwnd)

        s_left, s_top = win32gui.ClientToScreen(hwnd, (c_left, c_top))
        s_right, s_bottom = win32gui.ClientToScreen(hwnd, (c_right, c_bottom))

        client_rect = WindowRect(
            left=s_left,
            top=s_top,
            right=s_right,
            bottom=s_bottom,
        )

        if client_rect.width < MIN_CLIENT_WIDTH:
            return False, "连接失败：客户区宽度过小", None

        if client_rect.height < MIN_CLIENT_HEIGHT:
            return False, "连接失败：客户区高度过小", None

        dpi_scale = self._get_dpi_scale(hwnd)

        connected_window = ConnectedWindow(
            hwnd=hwnd,
            title=title,
            process_name=process_name,
            pid=current_pid,
            window_rect=window_rect,
            client_rect=client_rect,
            dpi_scale=dpi_scale,
        )

        return True, "", connected_window

    except Exception as exc:
        logger.exception("构建连接上下文异常")
        return False, f"连接失败：构建连接上下文异常：{exc}", None
```

## 15. 与 `connect_window()` 的集成

推荐集成方式：

```python
ok, error, connected_window = self._build_connected_window(window)
if not ok:
    self._last_error = error
    return ConnectResult.fail(error)
```

注意：

此处不应执行：

```python
self._connected_window = connected_window
```

保存必须等待 P0-06 激活成功。
